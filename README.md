## 🙋 FAQ

<details>
<summary>What is the live expression preview?</summary>

As you type a calculation expression, Calcbot shows the result updating in real time before you press equals — immediate feedback that catches entry errors.

</details>

<details>
<summary>Can I copy results from the calculation tape?</summary>

Yes. Any result in the tape copies to the clipboard for pasting into other applications.

</details>

<details>
<summary>Does Calcbot run on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs natively.

</details>

<details>
<summary>Who makes Calcbot?</summary>

Calcbot is made by Tapbots, known for thoughtfully designed Mac and iOS applications.

</details>

<details>
<summary>Can I continue calculating from a previous tape result?</summary>

Yes. Tapping any result in the tape selects it as the starting value for continued calculation.

</details>
